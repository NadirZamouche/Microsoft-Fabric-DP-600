# Microsoft Fabric Analytics Engineer DP-600  <img src="https://go-skill-icons.vercel.app/api/icons?i=fabric,pbi" height="30"/>

This repository contains **study notes, exam-focused explanations, and practice insights** for the **DP-600: Microsoft Fabric Analytics Engineer** certification.

The content follows the **official Microsoft exam skills outline** and focuses on **real Microsoft Fabric concepts** used in practice.

### Fabric Fundamentals & Governance

### Power BI & Semantic Models
- Power BI extensions (PBIX, PBIP, PBIR, PBIT, PBIDS)
- Data modeling & semantic models
- Power BI integration with Fabric
- Performance optimization & best practices

- OneLake & data organization
- OneLake catalog & Real-Time Hub
- Lakehouse, Warehouse, Eventhouse (KQL database)
- SQL Analytics Endpoint

### Data Ingestion & Transformation
- Data Pipelines
- Dataflows Gen2
- Notebooks
- Shortcuts & Database Mirroring

### SQL, Modeling & DAX
- T-SQL views, functions & stored procedures
- Type 1 & Type 2 Slowly Changing Dimensions (SCDs)
- Bridge tables
- DAX variables, iterators, table filtering
- DAX windowing & information functions
- Calculation groups, dynamic format strings & field parameters
